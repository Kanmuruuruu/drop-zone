# Start


### Front
yarn install

cd back

yarn install

cd ..

yarn start

This will start a react application on the port 3001


### Back

On an other terminal,
 
cd back

nodemon index.js

This will start a server.

## Use

then use the dropzone to upload in your server.

The Dropzone only accept jpg, jpeg, png and svg.
